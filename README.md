CartEtudiant – Petite Explication et README

Petite Explication
------------------
CartEtudiant est une petite application Android développée en Kotlin. Elle permet d’afficher une liste d’étudiants en utilisant un ListView/RecyclerView, ainsi qu’un spinner pour choisir ou filtrer des éléments. Le projet sert comme exercice pratique pour apprendre :
- la création d’adapters en Kotlin,
- l’affichage de données dans une liste,
- la gestion des layouts Android,
- l'organisation d’un projet Android (Models, Adapters, Activities).
